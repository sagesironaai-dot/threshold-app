Swarm agents. AGENTS.md states "Antigravity swarm agents operate under a separate namespace and identity registry" and defers conventions to SESSION_PROTOCOL.md. CLAUDE.md treats Antigravity as a single actor. If swarm nodes ever reach BUILD territory, the split-by-domain rule has no language for many-to-one handoff, and CLAUDE.md has no language for it at all.
Closing reconciliation rule. CLAUDE.md's "Gate closed" states that an unreconciled ROT_OPEN is not a valid close. Nothing in AGENTS.md acknowledges this. An agent reading AGENTS.md alone would not know that session closure has a reconciliation gate.
